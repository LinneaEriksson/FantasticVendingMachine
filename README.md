# FantasticVendingMachine

This is our fantastic vending machine! The project is built in C# and was a schoolproject.

# Installation

1.  You can clone this repo and open it with your favourite code-editor.
2.  Open the sln-file.
3.  Run the code.
4.  Enjoy!
